# ai-sop-setup v1.0.0

First public release of `ai-sop-setup`.

## Highlights

- Publishable Node.js CLI packaging with `bin`, `exports`, and release-safe file selection
- Multi-tool AI setup generation for Cursor, Claude Code, GitHub Copilot, Gemini CLI, OpenAI Codex, and Antigravity
- Support for third-party skills via the skills.sh ecosystem
- Strict config validation, `--dry-run`, non-interactive mode, and reusable config files
- Public-project readiness improvements including CI, issue templates, PR template, support docs, and bilingual README files

## Verification

- `npm run test:ci`
- `npm pack --dry-run`
