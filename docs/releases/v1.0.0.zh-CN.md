# ai-sop-setup v1.0.0

首个公开版本发布。

## 亮点

- 提供可发布的 Node.js CLI 包结构，包含 `bin`、`exports` 与精简发布文件清单
- 支持为 Cursor、Claude Code、GitHub Copilot、Gemini CLI、OpenAI Codex、Antigravity 生成项目级 AI 配置
- 支持接入 skills.sh 生态的第三方技能包
- 提供严格配置校验、`--dry-run`、非交互模式与可复用配置文件
- 补齐公开项目所需的 CI、Issue 模板、PR 模板、支持文档与中英文 README

## 验证结果

- `npm run test:ci`
- `npm pack --dry-run`
