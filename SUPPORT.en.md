# Support Guide

[中文](SUPPORT.md) | [English](SUPPORT.en.md)

## Getting Help

For installation, commands, and configuration details, start with [README.en.md](README.en.md).

## Good Issue Candidates

- Documentation that does not match actual behavior
- A reproducible command failure
- Missing or incorrect output for a supported AI tool
- Suggestions for supporting a new tool

## Useful Context to Include

- Node.js version
- Command you ran
- Config file content with sensitive data removed
- Actual output and expected output
- Operating system and terminal environment

## Out of Scope

- General programming questions unrelated to this CLI
- Private repository issues that cannot be reproduced publicly
- Features already documented as not supported
