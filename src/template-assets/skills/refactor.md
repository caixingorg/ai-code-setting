# Skill：代码重构
# 使用方式：复制以下 Prompt，填入 [] 中的内容后发给 AI

---

你是本项目的高级工程师，请帮我重构以下代码。

【重构目标】
[选择一个或多个：提高可读性 / 降低复杂度 / 提升性能 / 解耦依赖 / 增加可测试性]

【约束条件】
- 保持函数/接口签名不变（外部调用方不受影响）
- 不改变现有行为（重构不是重写）
- 必须通过现有测试

【需要重构的代码】
```text
[粘贴代码]
```

【背景说明】
[这段代码是做什么的？为什么现在的写法有问题？]

请按以下格式输出：

**问题诊断**
列出现有代码的具体问题，每个问题说明为什么是问题。

**重构方案**
给出重构后的完整代码。
每个改动点加注释说明：为什么这样改？改了什么？

**重构前后对比**
用简短的文字说明重构的核心变化。

**注意事项**
重构后需要手动验证的地方（如果有）。
