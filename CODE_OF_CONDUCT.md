# 社区行为准则

本项目希望保持专业、友好、可协作的交流环境。

## 我们鼓励

- 尊重不同背景与经验水平的参与者
- 以事实和实现细节讨论问题
- 对反馈保持具体、克制、可执行
- 在 review 中聚焦代码、文档与用户价值

## 我们不接受

- 人身攻击、羞辱、歧视或骚扰
- 恶意挑衅、刷屏、引战
- 泄露他人隐私、密钥或敏感信息
- 明知会破坏协作氛围的言行

## 执行方式

维护者可以对不符合本准则的内容进行编辑、隐藏、关闭或拒绝。
情节严重时，可限制参与权限。

## 适用范围

- Issue
- Pull Request
- 代码审查讨论
- 项目相关的文档与社区交流空间

## 报告问题

如遇协作行为问题，请通过维护者提供的私下联系方式处理；若仓库尚未配置，请先避免公开升级冲突，并等待维护者补充联系渠道。
